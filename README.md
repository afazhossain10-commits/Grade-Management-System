# Grade Management System

##  Overview
Java program to manage student marks and generate performance reports.

##  Features
- Add student marks
- Calculate average, highest, lowest
- Assign grades (A–F)
- Search student
- Generate report

##  How to Run
1. Compile: javac *.java
2. Run: java GradeManagementSystem

##  Structure
- src/ → source code
- docs/ → documentation
- examples/ → sample data
